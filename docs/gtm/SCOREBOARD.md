# Validation Scoreboard

**The product is frozen. This is the only board that moves.**

Update this — and nothing else — every week. Not screens added, not fields added, not
docs written, not commits. Those are *motion*, not progress. The bottleneck is no longer
the product; it is whether the market wants it.

---

## The build trigger — the most important rule in the project

> **The only sentence that authorizes new product work is: _"Show me that on our data."_**

| They say… | You do… |
|---|---|
| "Interesting." · "Nice idea." · "Cool." · "Makes sense." | **Nothing.** Keep talking. |
| "How do you prove that?" | Open the CFO Proof View → the case ledger. Still don't build. |
| **"Can you run this on our data?"** | **Build the leak assessment. _Now_ you build.** |

Curiosity is not demand. A request to touch their data is. Everything left of that
sentence is a conversation, not a commit.

---

## The board — update weekly, only this

| Signal | Count |
|---|---:|
| Conversations held | 0 |
| Asked to see the demo | 0 |
| Asked "how do you prove it?" | 0 |
| Asked to see it on **their** data | 0 |
| Pilot requests | 0 |
| Paying customers | 0 |

The funnel is intentional: each row is a stronger signal than the one above it. **Rows
4–6 are the only ones that justify writing code.** One genuine "run it on our data" is
worth more than twenty commits.

| Dimension | Score |
|---|---:|
| Problem hypothesis | 8 / 10 |
| Product narrative | 9 / 10 |
| Executive demo | 8.5 / 10 |
| Category clarity | 9 / 10 |
| Market proof | 2 / 10 |
| Paying-customer proof | 0 / 10 |

Conceptually mature; commercially unproven. That is not a flaw — it means the bottleneck
has moved **from the product to the market.**

---

## The order we hold — zealously

```
Conversation → Data → Pilot → Build Again
```

Never skip left to right. The biggest risk now is psychological, not technical: the
product is starting to *feel* real, and a real-feeling product is seductive to keep
polishing instead of testing whether anyone wants it. **Polishing is hiding. This
scoreboard is the cure.**
