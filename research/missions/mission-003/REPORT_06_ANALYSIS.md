## Disconfirmation & lineage (§8, §9 Phase 5)
- **Internal instability:** Gartner's own newsroom carries **multiple, rapidly-revised** AI-spend figures
  across 2025–2026 (e.g., $1.5T then $2.5T; GenAI $644B). The forecasts are **volatile and frequently
  restated** → a single number should never be quoted as settled. This is a freshness/precision finding, not
  a contradiction of a *measured* value (there is none — it's a forecast).
- **Lineage (§8):** press outlets repeating "$644B" **descend from the one Gartner release** = **one lineage**.
  No independent market-sizing corroborates the exact figure; competing houses (IDC, etc.) publish *different*
  numbers, underscoring model-dependence.
- **False-precision flag:** "**76.4%**" and "**$644B**" present three-significant-figure precision for a
  multi-year forecast — NH-ROP §7 (methodological transparency + completeness) caps confidence **Low** absent
  a disclosed model and error band.

## Cross-examination
- **Forecast discipline (§5/§12):** C1–C3 are *opportunity/forecast* figures, never realized value. Quoting

- **Authority bias (§19):** "Gartner says" is a prior, not a verdict. NH-ROP treats the brand as source-tier
